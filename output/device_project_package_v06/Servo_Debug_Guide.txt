Servo Debug Guide - V0.7.1
================================================================================

一、调试前确认
1. 确认伺服驱动器参数已正确设置。
2. 确认正负限位、原点信号、急停、安全回路有效。
3. 确认机械机构无干涉。
4. 首次运行必须低速点动。

二、推荐调试顺序
1. Servo On
2. 正反向点动，确认方向
3. 回零
4. 小距离绝对定位
5. 点位表逐点验证
6. 与气缸/工站流程联动
7. 急停和报警复位测试

三、当前轴清单
- Axis[1] AX1 / X轴
  - P1: 上料位 = 0.0 mm
  - P2: 扫码位 = 120.0 mm
  - P3: 下料位 = 300.0 mm

四、重要说明
本版本生成的是伺服控制框架，不替代真实运动控制调试。
需要 PLC 工程师在 Sysmac Studio 中绑定真实 Axis 变量，并接入 MC_Power、MC_Home、MC_MoveAbsolute、MC_Stop 等运动控制指令。
